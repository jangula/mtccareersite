## Section 5: Key User Flows

### Flow 1: Job Application Process
1. Browse jobs (`/jobs`) - See `02_jobs_listing_desktop.png`
2. Select a job - See `03_job_detail_desktop.png`
3. Click "Apply Now" - See `04_application_form_desktop.png`
4. Fill form, select gender/race
5. Check privacy consent checkbox
6. Submit application

### Flow 2: HR Review Process
1. HR Dashboard - See `12_admin_dashboard_desktop.png`
2. Click application - See `17_admin_application_detail_desktop.png`
3. Review candidate info
4. Click "Shortlist" - Status updated
5. Automatic email sent to candidate

### Flow 3: Analytics Review
1. HR Dashboard - Click "Analytics"
2. View overall metrics - See `13_admin_analytics_desktop.png`
3. Check gender/race demographics
4. Review hire demographics
5. Export insights (future feature)
